Shortcuts

  • Space to play or pause the automatic data browsing
  • Backspace to reset the current line to 0
  • Left and Right to Navigate the X axis
  • Up and Down to change de Y axis field
  • P to hear the values at the current position
  • R to hear the minimum and maximum values on the Y axis
  • C to hear the number of columns and rows in the opened dataset
  • Escape to interrupt the current speech
  • M to mute or unmute the beeping
  • L to switch the current language
  • I to hear general informations

Raccourcis

  • Espace pour lancer ou arrêter la navigation automatique des données
  • Retour arrière pour remettre la ligne actuelle à 0
  • Gauche et Droite pour naviguer sur l'axe des abscisses
  • Haut et bas pour changer le champ utilisé sur l'axe des ordonnées
  • P pour écouter les valeurs à la position actuelle
  • R pour écouter les valeurs minimales et maximales sur l'axe des ordonnées
  • C pour écouter le nombre de colonnes et de lignes dans l'ensemble de données ouvert
  • Échap pour interrompre la voix en cours
  • M pour activer ou désactiver le mutisme du bip
  • L pour changer le langage
